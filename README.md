# iNatSpectro – Support & Community

This repository is the place to get help, report issues, suggest features, or contribute to [iNatSpectro](https://www.inatspectro.org) (the Chrome extension for iNaturalist spectrum views).

- **Bug reports** – [Open an issue](https://github.com/iNatSpectro/support/issues/new?template=bug_report.md) and choose the bug report template.
- **Feature requests / ideas** – Use the [feature request template](https://github.com/iNatSpectro/support/issues/new?template=feature_request.md).
- **Questions / help** – [Open a general issue](https://github.com/iNatSpectro/support/issues/new).

The main extension code lives in a separate private repository; this repo is for community interaction only.
